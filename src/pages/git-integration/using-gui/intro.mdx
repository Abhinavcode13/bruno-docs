# **Collaboration via GUI** <PremiumBadge />

Bruno offers an intuitive **GUI-based Git collaboration** feature, available in the [Pro](https://www.usebruno.com/buy?plan=pro) and [Ultimate Editions](https://www.usebruno.com/buy?plan=ultimate). This feature makes it easier for both engineering and non-engineering teams to collaborate seamlessly on collections.

<Callout type="warning" emoji="⚠️">
You need to have Git installed on your machine to use these feature.
</Callout>

This feature includes following actions:

- [Initializing Git in a Collection](../using-gui/provider#initializing-git)
- [Adding and Committing Changes](../using-gui/provider#adding-and-committing-changes)
- [Pushing and Pulling Changes](../using-gui/provider#pushing-and-pulling-changes)
- [Branch Creation and Checkout](../using-gui/provider#branch-creation-and-checkout)
- [Cloning a Repository](./consumer#cloning-collections-from-github)
